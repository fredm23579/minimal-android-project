# 📱 Minimal Android Project

A lean, foundational repository demonstrating the bare-metal essentials of a valid, functional Android application. This project strips away the boilerplate typically generated by IDEs to show the core architecture of an Android app.

[![Build](https://img.shields.io/badge/Build-Gradle-green)](https://gradle.org)

## 🎯 Purpose

This repository explores the absolute minimum requirements to set up and build a working Android project from scratch. It serves as an educational resource for understanding the underlying build process and project structure without the noise of modern IDE templates.

## 🛠️ Core Components

To build this project, you only need:
- **One `.java` activity source file**: Placed in its appropriate package directory.
- **`AndroidManifest.xml`**: Defining the app's structure and entry point.
- **`build.gradle`**: Simple build configuration (optional if using manual SDK tools, but recommended for simplicity).

## 💻 Tech Stack

- **Language**: Java 8+
- **Build System**: Gradle 8.7+
- **Android SDK**: Platform 34+ (configurable in `build.gradle`)
- **Plugin**: Android Gradle Plugin (AGP)

## 🚀 Getting Started

### Prerequisites

Ensure your environment is configured with the following:
- **Java**: `JAVA_HOME` set and `java` in your PATH.
- **Android SDK**: `ANDROID_HOME` pointing to your SDK root.
- **Gradle**: `gradle` 8.7 or higher installed.

### Build & Install

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/minimal-android-project.git
   cd minimal-android-project
   ```
2. Build and install to a connected device or emulator:
   ```bash
   gradle installDebug
   ```

The application will be deployed to all devices accessible via `adb`.

## 📂 Project Structure

Following the standard Gradle Android plugin convention:

```text
minimal-android-project/
├── build.gradle          # Build configuration
└── src/
    └── main/
        ├── AndroidManifest.xml   # App manifest
        └── java/
            └── pl/czak/minimal/  # Source code
                └── MainActivity.java
```
